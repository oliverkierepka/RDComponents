# RDComponents

This is the monorepo for RD Web-Components. 

[![lerna](https://img.shields.io/badge/maintained%20with-lerna-cc00ff.svg)](https://lernajs.io/)

Getting Started
---------------
```bash
$ npm install
$ npm build
$ npm start
```

Testing
-------
```bash
$ npm run test
```

Linting 
-------
```bash
$ npm run lint
```
